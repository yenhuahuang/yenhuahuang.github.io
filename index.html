WiCare system (此專案成果被留存以供日後課程參考)

專案內容

為強化國內醫療系統與疾病管制署之間對傳染性疾病的信息傳遞效果，在此系統，醫院方可以主動的去判斷並且做疾病的檢驗，再通報管制署以作相關配套措施，有別於今日以管制署為主動方的送驗機制。

使用工具/技術

Java Swing, SQL developer, OpenCV
